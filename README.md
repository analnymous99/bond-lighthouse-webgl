# Кот Бонд и Маяк историй

Веб-игра для геймджема. Игрок помогает коту Бонду восстановить свет Маяка, проходя сюжетные задания с выбором, текстовым вводом и загрузкой рисунка.

## Запуск

Открыть опубликованную WebGL-сборку по ссылке GitHub Pages:

https://analnymous99.github.io/bond-lighthousw-webgl/

## Управление

- Нажимайте кнопки выбора в сюжетных сценах.
- В текстовых заданиях вводите короткий ответ.
- В заданиях с рисунком загрузите изображение через кнопку загрузки.

## Технологии

- Unity
- WebGL
- GitHub Pages
